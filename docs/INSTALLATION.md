## Installation

This project can be installed many ways but here we'll provide some ways

### First-time installation

```sh
curl -s https://raw.githubusercontent.com/dalisoft/lint-staged-sh/master/lint-staged.sh | sh
```

### Afterwards command

```sh
./lint-staged.sh
```
